<p>This release is entirely made up of improvements to the build process and is being released in order to test that process. There are no user-facing bug fixes or improvements in this release as compared to 3.13.0.</p>

<p>This release includes code contributions from <a href="https://github.com/CharliePoole">Charlie Poole</a> and <a href="https://github.com/mikkelbu">Mikkel Nylander Bundgaard</a>.</p>

<p>You can download the NUnit Console from <a href="https://github.com/nunit/nunit-console/releases">GitHub</a>, <a href="https://www.nuget.org/">NuGet</a> or <a href="https://www.chocolatey.org/profiles/nunit.org">Chocolatey</a>. See the <a href="https://docs.nunit.org/articles/nunit/release-notes/console-and-engine.html">release notes</a> for more information.</p>

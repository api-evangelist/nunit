<p>This release addresses several misses with the new <code class="language-plaintext highlighter-rouge">FixtureLifeCycle</code> attribute, switches to using <a href="https://github.com/dotnet/sourcelink">SourceLink</a> and NuGet <a href="https://docs.microsoft.com/en-us/nuget/create-packages/symbol-packages-snupkg">snupkg</a> packages for debugging into NUnit from your unit tests. It also addresses issues with the time format of ignored and explicit tests in the test results file.</p>

<p>You may download NUnit 3.13.1 from <a href="https://github.com/nunit/nunit/releases">Github</a>. See the <a href="https://docs.nunit.org/articles/nunit/release-notes/framework.html">release notes</a> for more information.</p>

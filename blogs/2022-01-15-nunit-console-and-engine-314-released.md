<p>This release features a new agent for tests targeting .NET 5.0. In addition, automation of our publication and release process is now complete from creation of a draft release through releasing to production on GitHub. While this doesn’t impact users directly, it will allow us to speed up the introduction of new features in coming releases.</p>

<p>This release includes code contributions from <a href="https://github.com/CharliePoole">Charlie Poole</a>, <a href="https://github.com/jnm2">Joseph Musser</a>, <a href="https://github.com/mikkelbu">Mikkel Nylander Bundgaard</a> and <a href="https://github.com/k15tfu">Ilia K</a>. Thank you to all those who contributed both in code, and otherwise.</p>

<p>You can download the NUnit Console from <a href="https://github.com/nunit/nunit-console/releases">GitHub</a>, <a href="https://www.nuget.org/">NuGet</a> or <a href="https://www.chocolatey.org/profiles/nunit.org">Chocolatey</a>. See the <a href="https://docs.nunit.org/articles/nunit/release-notes/console-and-engine.html">release notes</a> for more information.</p>
